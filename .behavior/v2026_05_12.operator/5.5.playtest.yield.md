## pass/fail criteria

- **pass if:**
  - all 33 unit tests pass
  - `npx rhachet roles list --repo ghlitch` shows observer, operator, deployer with correct skills
  - role registry contains exactly 7 roles

- **fail if:**
  - any unit test fails
  - any role absent from registry
  - skill directories not configured for any role
