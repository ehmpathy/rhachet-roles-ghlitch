## contract delivered

```
rhx provision.database --which livedb --env <env> --mode plan
rhx provision.database --which livedb --env <env> --mode apply
rhx provision.database --which livedb --env <env> --mode sync --slug <change-slug>   # new
```

- exit 0 = done; exit 1 = malfunction; exit 2 = constraint (bad args / bad slug-mode combo / blocked prod write)
- prod: apply + sync gated by `provision.uses`; plan open
