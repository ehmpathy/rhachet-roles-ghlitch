# Project Context Instructions

## Context Loading

**IMPORTANT**: Always check the `.briefs/` directory at the start of each conversation to load project context, requirements, and documentation.

The `.briefs/` directory contains:
- Project requirements and specifications
- Design documents
- Context about the codebase
- Guidelines and constraints

Before starting any task, read relevant files from `.briefs/` to understand the current project context and ensure your work aligns with project requirements.
