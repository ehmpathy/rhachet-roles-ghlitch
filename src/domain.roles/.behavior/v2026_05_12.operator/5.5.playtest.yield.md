# 5.5.playtest.yield

## summary

playtested the two-header mascot/artifact pattern across all ghlitch role skills.

## verified

all skills now use consistent output format:
- mascot header: `🐈 {vibe phrase}` + blank line
- artifact header: `{artifact} {skill-name} [--flags]` + tree body

vibe phrases follow standard cat personality:
- `lets prowl...` = plan/help
- `purrfect!` / `caught it!` = success
- `hiss...` = blocked (malfunction)
- `claws out...` = constraint
