# ghlitch roles

roles for observability, deployment, diagnostics, budgets, health, and alerts.

## 🔮 observer

watch all that goes on — surfaces anomalies and tracks patterns.

## ⛵ deployer

escort code to prod safely across environments.

## 🔍 detective

reactively bisect to find root causes fast.

## 💰 budgeter

track infrastructure costs to ensure stability.

## 🔔 alerter

ensure real defects alert while false positives stay silent.

## 🛡️ hardener

review for security vulnerabilities and harden systems.
