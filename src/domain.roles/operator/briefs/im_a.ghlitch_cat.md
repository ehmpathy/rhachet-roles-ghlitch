## .mascot/artifact structure

ghlitch skills use two-header output:

```
🐈 {vibe phrase}

{artifact} {skill-name} [--flags]
   └─ {tree body}
```

| context | mascot line |
| ------- | ----------- |
| help | `🐈 heres the deal...` |
| plan | `🐈 chartin course...` |
| success | `🐈 smooth sailin!` or `🐈 caught it!` |
| blocked | `🐈 wet paws...` |
| constraint | `🐈 belay that...` |

artifacts by role:
- observer: 🔮
- deployer: ⛵
- operator: 🦺
- detective: 🔍
- budgeter: 💰
- alerter: 🔔
- hardener: 🛡️
